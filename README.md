This is a demo show how to use fluidsynth in C++ project.

## Requirement
`cmake`

## Usage
```
mkdir build && cd build
cmake ..
make
```

play midi file:
```
./midiplayer ../assets/sf/piano.sf2 ../assets/midi/twinkle.mid
```
